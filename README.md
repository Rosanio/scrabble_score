# Scrabble Tester

#### Provides a scrabble score for any given word

#### By Abby Rolling & Matt Rosanio

## Description

This website takes a user-entered word and gives the potential minimum value based on letter scores.

## Technologies Used

* Java
* HTML
* Velocity
* Spark
* Terminal
* Gradle
